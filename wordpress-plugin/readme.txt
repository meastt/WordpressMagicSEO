Connects your WordPress site to the Magic SEO Vercel automation platform.

== Description ==

This plugin provides a secure way to connect your WordPress site to the Magic SEO Vercel automation server. 
It allows you to store the necessary credentials (Username and Application Password) that the automation server needs to analyze and improve your site's SEO.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/magic-seo-connector` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->Magic SEO screen to configure the connection.

== Frequently Asked Questions ==

= Where do I get an Application Password? =

Go to Users > Profile, scroll down to "Application Passwords", give it a name (e.g., "Magic SEO"), and copy the generated password.
