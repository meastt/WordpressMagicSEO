AI-powered SEO content optimization with Claude, Gemini, and advanced affiliate management.

== Description ==

Magic SEO is a premium WordPress plugin that uses AI to optimize your content for better search rankings.

**Features:**

* **AI Content Optimization** - Use Claude and Gemini to improve titles, meta descriptions, and content
* **Affiliate Link Manager** - Manage affiliate IDs for Amazon, ShareASale, CJ, Impact, and more
* **AI Link Checker** - Scan posts for broken links and missing affiliate tags
* **Secure API Storage** - Encrypted storage for all API keys and credentials
* **Beautiful Dashboard** - 2026-designed glassmorphic admin interface

== Installation ==

1. Upload the `magic-seo` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Magic SEO in the admin menu
4. Add your API keys in Settings
